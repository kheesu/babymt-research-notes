# Tags — taxonomy

The controlled vocabulary for this vault. Keep tags to this set so search and
Dataview stay predictable.

## Note type
- `#moc` — map of content
- `#finding` — empirical result
- `#decision` — design decision (ADR)
- `#run` — a training/eval run or run group
- `#literature` — paper / prior work
- `#journal` — dated session log
- `#meta` — vault machinery

## Phase
- `#phase/1` `#phase/2` `#phase/3` `#phase/4` `#phase/5`

## Condition
- `#condition/c0` `#condition/c1` `#condition/c2` `#condition/c3` `#condition/c4`

## Language
- `#lang/en` `#lang/zh` `#lang/ja` `#lang/ko`

## Status (also a frontmatter key)
- `#status/stub` `#status/draft` `#status/stable` `#status/superseded`
- workflow extras: `#status/running` `#status/blocked` `#status/planned`

## Paper
- `#paper/claim` — goes into the paper as a claim
- `#paper/limitation`
- `#paper/figure` `#paper/table`
